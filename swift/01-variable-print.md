# 變數

-   可變動

```swift
var apple:String = "apple"
```

-   變數，不可變動

```swift
let banana:String = "banana"
```

-   列印

```swift
print("Hello",apple) // Hello apple
print("Hello \(apple)") // Hello apple
print("Hello",apple,separator:"---") // Hello---apple
print("Hello",apple,separator:"---",terminator:"!!!") // Hello---apple!!!
```
