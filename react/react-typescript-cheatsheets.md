## 引入 React

```js
import * as React from 'react';
import * as ReactDOM from 'react-dom';
```

如果 tsconfig.json 有啟用 `allowSyntheticDefaultImports": true`

```js
import React from 'react';
import ReactDOM from 'react-dom';
```
